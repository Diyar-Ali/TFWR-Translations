# タプル
タプルは複数の値を一つにまとめるのに便利だよ。
タプルを作るには、値をカンマで区切るだけだよ：

`tuple = 1, 2`

また、タプルを再度複数の変数に展開することもできるよ。以下のコードでは、タプル`(1,2)`が変数`a`と`b`に展開されてるよ。

`a, b = 1, 2`

タプルはリストのようにインデックスでアクセスできるけど、一度作成すると変更できないよ（不変だよ）。

`tuple = 1, 2`

`print(tuple[1])`
は`2`を表示するよ

`tuple[0] = 3`
はエラーが発生するよ

リストとは違って、タプルはdictionaryのキーとして使うことができるよ。

`d = {(1,2):(4,5)}

print(d[(1,2)])`
